# nexu v0.1.0: The First Public Release Is Here

> The first public release of nexu — a double-click-install OpenClaw desktop client with built-in Skills, top-tier models, and BYOK support.

The first public release of **nexu** — The Simplest OpenClaw Desktop for OPC.

## Highlights

**Double-click install** — Download, launch, and start using AI agents in seconds. No terminal, no config files.

**Built-in OpenClaw Skills + full Feishu Skills** — Native skill integrations ship out of the box, plugging directly into your team's existing workflows.

**Top-tier models, ready to go** — Access Claude 4.6, ChatGPT 5.4, Minimax 2.5, GLM 5.0, Kimi 2.5, and more via your Nexu account.

**Bring your own API Key** — Use your own model providers without creating an account or logging in.

**Desktop client for macOS** — A native Electron-based desktop experience built for real daily use.

## What's Included

Web frontend + Backend API + Landing page. Desktop client (macOS, Apple Silicon). Gateway & Router services. OpenClaw core runtime. Built-in skill system (local dev & bot skills). Deployment configs (Helm, K8s, Nginx).

## System Requirements

OS: macOS 12+ (Apple Silicon). Storage: ~500 MB.

Windows & macOS Intel support is in development.

## Get Started

Download the Mac client from this release, or visit [nexu.io](https://nexu.io) for more information.
