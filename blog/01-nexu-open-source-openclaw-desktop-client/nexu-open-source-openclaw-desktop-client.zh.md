> 开源桌面客户端，让你的 OpenClaw Agent 在微信、飞书、Slack、Discord 里直接运行。双击安装、自带 Key、数据本地优先。公测期间顶级模型完全免费。

## nexu（奈苏）是什么？

**nexu**（奈苏，next to you）是一个开源桌面客户端，让你的 **OpenClaw 🦞** Agent 直接运行在微信、飞书、Slack、Discord 等 IM 中。

**已支持微信接入 OpenClaw** —— 适配微信 8.0.7 OpenClaw 插件，点击连接、微信扫码，即可在微信中与 AI Agent 对话。

下载即用，图形化配置，内置飞书 Skills，支持 Claude / GPT / Gemini 等多模型与自带 API Key。

连接 IM 后，Agent 7×24 小时在线——手机上随时对话，不受桌面限制。所有数据保存在本机，隐私安全，完全可控。

## 与常见方案的区别

**模型：**OpenClaw 官方自选但需手动配置。托管方案平台指定不可更换。nexu 自选 Claude / GPT / Gemini 等，GUI 一键切换。

**数据路径：**OpenClaw 本地。托管方案经第三方服务器，数据不可控。nexu 本机为主，不托管你的业务数据。

**费用：**OpenClaw 免费但需自行部署。托管方案订阅/按席收费。nexu 客户端免费，按自备 API Key 计费。

**源码：**OpenClaw 开源。托管方案闭源无法审计。nexu MIT 开源，可 fork、可审计。

**渠道：**OpenClaw 需自行对接。托管方案视产品而定常有限制。nexu 内置微信、飞书、Slack、Discord，开箱即用。

**使用方式：**OpenClaw 是 CLI 需技术背景。nexu 纯 GUI，无需 CLI，双击即用。
