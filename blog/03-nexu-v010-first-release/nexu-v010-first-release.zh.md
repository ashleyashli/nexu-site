# nexu v0.1.0：首个公开版本正式发布

> nexu 首个公开版本正式发布——双击安装即可使用的 OpenClaw 桌面客户端，内置丰富 Skills 和主流大模型。

**nexu** 首个公开版本——最简单的 OpenClaw 桌面客户端，面向 One Person Company。

## 亮点

**双击安装**——下载、启动，几秒内开始使用 AI Agent。无需终端，无需配置文件。

**内置 OpenClaw Skills + 完整飞书 Skills**——原生技能集成开箱即用，直接对接团队现有工作流。

**顶级模型，即开即用**——通过 nexu 账号直接使用 Claude 4.6、ChatGPT 5.4、Minimax 2.5、GLM 5.0、Kimi 2.5 等。

**支持自带 API Key**——使用自己的模型供应商，无需注册、无需登录。

**macOS 桌面客户端**——基于 Electron 的原生桌面体验，为日常使用而生。

## 包含内容

Web 前端 + 后端 API + Landing 页面。桌面客户端（macOS，Apple Silicon）。网关 & 路由服务。OpenClaw 核心运行时。内置技能系统（本地开发 & 机器人技能）。部署配置（Helm、K8s、Nginx）。

## 系统要求

系统：macOS 12+（Apple Silicon）。磁盘：约 500 MB。

Windows 与 macOS Intel 支持开发中。

## 开始使用

从本次发布下载 Mac 客户端，或访问 [nexu.io](https://nexu.io) 了解更多。

完整变更日志：[github.com/nexu-io/nexu/commits/v0.1.0](https://github.com/nexu-io/nexu/commits/v0.1.0)
